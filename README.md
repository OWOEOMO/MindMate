# MindMate — Vibe Coding Challenge（前端展示）

這是為 **JUMPSTARTER ZPIRE** Round 1 的 *Vibe Coding Challenge* 準備的前端落地頁（Landing Page）。
- 純前端：HTML / CSS / JS（無框架、無後端）。
- RWD：行動裝置友善。
- 一鍵部署 GitHub Pages。

## 快速開始
1. 直接打開 `index.html`。
2. 或在專案根目錄啟動本地伺服器（任一方式皆可）：
   - Python：`python -m http.server 8080`
   - Node：`npx http-server -p 8080`

## 部署到 GitHub Pages
1. 建立 GitHub Repository（公開）。
2. 將本專案所有檔案推送到 `main` 分支根目錄。
3. 進入 **Settings → Pages**：
   - **Source** 選擇 `Deploy from a branch`
   - Branch 選 `main`、資料夾選 `/root`，按 **Save**。
4. 等待數十秒，右上角會出現可公開存取的網址。

> 評審可直接於瀏覽器開啟你的 GitHub Pages 網址檢視頁面。
