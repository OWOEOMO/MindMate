# MindMate — Vibe Coding Challenge (Frontend Demo)

This is a frontend landing page prepared for **JUMPSTARTER ZPIRE** Round 1 *Vibe Coding Challenge*.
- Pure frontend: HTML / CSS / JS (no frameworks, no backend).
- RWD: Mobile-friendly design.
- One-click deployment to GitHub Pages.

## Quick Start
1. Open `index.html` directly.
2. Or launch a local server in the project root (any method works):
   - Python: `python -m http.server 8080`
   - Node: `npx http-server -p 8080`

## Deploy to GitHub Pages
1. Create a public GitHub repository.
2. Push all project files to the root of the `main` branch.
3. Go to **Settings → Pages**:
   - Under **Source**, choose `Deploy from a branch`.
   - Select branch `main`, folder `/root`, then click **Save**.
4. Wait a few seconds; a public URL will appear in the upper right corner.

> Judges can open your GitHub Pages URL directly in a browser to view the page.
